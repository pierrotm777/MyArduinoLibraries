Au total, le programme se compose de 5 modules de code.
- SK33v2.ino est la fonction principale (SK33 vient du nom de construction du pari sur glace de la classe 20m)
- SK33RcvData est le stockage des données reçues
- FrSkySportTelemetryReader sont du code et des en-têtes pour lire les données entrées via la télémétrie
- SK33AccessPoint est... eh bien, son nom l'indique, le point d'accès WiFi qu'ouvre l'ESP8266
- SK33WebServer dit aussi de quoi il s'agit, le serveur web qui reprend la page sur le smartphone ou la tablette
